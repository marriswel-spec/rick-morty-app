# 🛸 Rick & Morty App

## 📌 Descripción
Aplicación web desarrollada con **React + Vite** que consume la API oficial de Rick & Morty.  
Permite explorar personajes, ver detalles, filtrar resultados y gestionar una lista de favoritos.

---

## 🚀 Funcionalidades

✅ Listado de personajes  
✅ Paginación  
✅ Búsqueda / filtro por nombre  
✅ Visualización de personaje individual  
✅ Sistema de favoritos ❤️ (persistente con localStorage)  
✅ Navegación entre páginas (React Router)  
✅ Diseño moderno, responsive y modo oscuro  

---

## 🛠️ Tecnologías utilizadas

- ⚛️ React
- ⚡ Vite
- 🌐 Axios (consumo de API)
- 🔀 React Router DOM
- 🎨 CSS moderno (Flexbox / Grid / Glassmorphism)

---

## 📡 API utilizada

Rick & Morty API
